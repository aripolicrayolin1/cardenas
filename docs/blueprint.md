# **App Name**: GuardianCultivo

## Core Features:

- Monitoreo de Sensores en Tiempo Real: Muestra datos en vivo de los sensores IoT conectados, incluyendo humedad del suelo, temperatura y radiación UV, en un panel interactivo.
- Alertas Predictivas de Riesgo: Una herramienta impulsada por IA que analiza datos de sensores en tiempo real para predecir y notificar a los usuarios sobre condiciones propensas a plagas o enfermedades fúngicas específicas.
- Diagnóstico de Cultivos impulsado por IA: Permite a los agricultores subir fotos desde sus dispositivos móviles para un análisis inmediato de IA que identifique posibles enfermedades o plagas en los cultivos.
- Red de Alerta Comunitaria: Un sistema para enviar alertas automáticas y localizadas a agricultores registrados cercanos en Hidalgo sobre brotes confirmados de enfermedades o infestaciones de plagas.
- Gestión de Fincas y Sensores: Los agricultores pueden registrar sus campos individuales, definir límites y asociar sensores IoT específicos a cada área para un monitoreo dirigido.
- Autenticación y Perfiles de Usuario: Registro de usuario seguro y gestión de perfiles para agricultores, permitiéndoles administrar su información personal y la ubicación de sus fincas.

## Style Guidelines:

- Color principal: Un verde equilibrado y natural (#339933) que representa el crecimiento y la precisión agrícola.
- Color de fondo: Un verde claro y altamente desaturado (#EFF5EF) para proporcionar una base limpia y fresca.
- Color de acento: Un amarillo vibrante (#FFF233) para elementos de alto contraste, alertas críticas y llamadas a la acción clave, asegurando una atención inmediata.
- Fuente para el cuerpo y los titulares: 'Inter' (sans-serif) por sus características modernas, limpias y altamente legibles, adecuadas para la visualización de datos y alertas.
- Utilizar iconos de línea claros y modernos que representen intuitivamente elementos agrícolas, condiciones meteorológicas, tipos de sensores y alertas comunitarias.
- Adoptar un diseño de panel limpio y modular centrado en la visibilidad de datos en tiempo real. Optimizar para uso móvil primero, permitiendo interacciones fáciles en el campo y cargas de fotos.
- Implementar animaciones sutiles y con propósito para las actualizaciones de datos en los paneles, la confirmación de acciones exitosas (por ejemplo, carga de fotos, alerta enviada) y transiciones suaves entre secciones.
